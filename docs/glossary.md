# Glossary

One-line definitions, cross-linked to the page that derives each.

- **MDP** — $(\mathcal S,\mathcal A,P,r,\gamma)$; the formal decision problem. [01](00_foundations/01_mdp_and_returns.md)
- **Return $G_t$** — discounted sum of future rewards; what we maximize. [01](00_foundations/01_mdp_and_returns.md)
- **Markov property** — future depends on the past only through the present state. [01](00_foundations/01_mdp_and_returns.md)
- **Value $V^\pi$ / $Q^\pi$** — expected return from a state / state-action under $\pi$. [01](00_foundations/01_mdp_and_returns.md)
- **Advantage $A=Q-V$** — how much better an action is than average; PPO's signal. [02](00_foundations/02_policy_gradients.md)
- **Bellman equation** — recursive consistency of value functions. [01](00_foundations/01_mdp_and_returns.md)
- **Log-derivative trick** — $\nabla\log f=\nabla f/f$; turns ∇𝔼 into 𝔼∇. [02](00_foundations/02_policy_gradients.md)
- **Policy gradient theorem** — gradient as 𝔼[∇log π · advantage]; no dynamics gradient. [02](00_foundations/02_policy_gradients.md)
- **Baseline** — action-independent term subtracted to cut variance, unbiased. [02](00_foundations/02_policy_gradients.md)
- **TD error $\delta_t$** — $r+\gamma V(s')-V(s)$; one-step advantage estimate. [03](00_foundations/03_variance_and_gae.md)
- **GAE($\lambda$)** — exp-weighted sum of TD errors; bias↔variance dial. [03](00_foundations/03_variance_and_gae.md)
- **Truncation vs termination** — timeout (bootstrap) vs real end (don't). [03](00_foundations/03_variance_and_gae.md)
- **Importance ratio $r_t(\theta)$** — $\pi_\theta/\pi_{\text{old}}$; enables data reuse. [04](00_foundations/04_ppo.md)
- **Clipped surrogate** — PPO's objective; removes incentive to step too far. [04](00_foundations/04_ppo.md)
- **Entropy bonus** — rewards a wide action distribution; sustains exploration. [04](00_foundations/04_ppo.md)
- **Adaptive-KL LR** — scale LR to hit a target policy KL (rsl_rl). [04](00_foundations/04_ppo.md)
- **Gaussian policy** — continuous actions via $\mathcal N(\mu_\theta(s),\sigma)$, learned log-std. [05](00_foundations/05_continuous_control.md)
- **tanh-squash Jacobian** — log-prob correction for bounded actions. [05](00_foundations/05_continuous_control.md)
- **PD position targets** — actions as joint offsets through a PD loop; safer exploration. [05](00_foundations/05_continuous_control.md)
- **Asymmetric actor-critic** — critic sees privileged state the actor can't. [30](30_teacher/README.md)
- **Terrain curriculum** — promote/demote difficulty by walked distance. [30](30_teacher/README.md)
- **Domain randomization** — train across a distribution of physics for robustness. [30](30_teacher/README.md)
- **Privileged information** — height-scan / true velocity available only in sim. [30](30_teacher/README.md)
- **DAgger** — roll out the student, supervise with the teacher; fixes covariate shift. [40](40_distillation/README.md)
- **Belief state** — recurrent hidden state as an estimate of hidden world state. [40](40_distillation/README.md)
